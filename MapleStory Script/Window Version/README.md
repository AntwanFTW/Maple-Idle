SOLO KPQ Automation Script v1.1

MapleStory (BlueStacks) – Image Recognition + Keyboard Automation

** What this does**

This Python script automates SOLO KPQ queueing by:

Detecting on-screen buttons using image recognition

Pressing mapped keyboard keys automatically:

Enter → F6

Accept → F9

OK → F7

Priority order:

Enter (highest)

Accept

OK (lowest)

Troubleshooting Tips:
If buttons are not recognized, screenshot icons for Enter, Accept and OK and replace it in the script folder
Adjust Confidence number between 0.5 to 1. (Lower the number, the less accurate it has to be to recognize the image its looking for)
